# The Noise Wall Lie
## Highway Noise Barrier Effectiveness & Equity · Los Angeles

**Live Demo:** https://kalchikee.github.io/noise-wall-lie/

---

### Overview

State DOTs spend over **$500 million annually** on highway noise barriers. These walls are justified by acoustic engineering models promising 10–15 dB reductions. But are those models accurate? This project tests whether LA's noise barriers deliver their promised reductions — and whether barrier installation follows demographic patterns.

### Key Findings

- **Average promised reduction:** 10.1 dB | **Average actual:** 8.4 dB — a systematic 1.7 dB performance gap
- Neighborhoods over 75% white receive barrier protection at **2.4x the rate** of majority-minority communities on the same highways
- **8 documented gaps** in the LA highway noise barrier network, disproportionately located in lower-income communities
- Q1 income neighborhoods (under $40k): only **28% protected** vs. Q4 (over $85k): **74% protected**
